# EVAL — Tactical Performance Assessment

![EVAL Platform Hero](https://images.unsplash.com/photo-1550745165-9bc0b252726f?auto=format&fit=crop&q=80&w=2070)

**EVAL** is a high-fidelity, industrial-grade performance assessment platform designed with a **Tactile Void** aesthetic. It provides a brutalist yet sophisticated interface for evaluating cognitive skills, motor precision, and strategic thinking through a series of specialized modules.

## 🕹️ Platform Philosophy

Built for the "Industrial Arcade," EVAL rejects generic, colorful gamification in favor of a high-contrast, data-driven environment. Every pixel, scanline, and transition is engineered to create a sense of focused urgency and technical precision.

## ✨ Core Features

### 1. Mission Debriefing Portal
A compact, high-impact completion interface that provides immediate performance feedback.
- **Dynamic Best Score Tracking**: Automatically persists and compares metrics via localized storage.
- **Tactile Visuals**: Featuring animated Pixel Trophies and industrial status badges.
- **Zero-Clutter Design**: Stripped of decorative elements to focus purely on mission data.

### 2. Tactical Profile Dossier
A comprehensive user overview module designed as a high-tech digital portfolio.
- **Animated Pacman Avatar**: A custom-engineered, pixelated "Tactical Pacman" with synchronized chomping logic and moving power-dot interaction.
- **Unified Stats Engine**: Real-time visualization of global rank, accuracy, play time, and assessment totals.
- **Box-less Industrial Grid**: A minimalist layout that flows directly onto the background, emphasizing typography and data integrity.

### 3. Specialized Assessment Modules
- **Grid Challenge**: High-precision spatial recognition.
- **Switch Challenge**: Heuristic permutation and logic processing.
- **Motion Challenge**: Viewport-triggered sequential logic.

## 🛠️ Tech Stack

- **Core**: React 18 + TypeScript
- **Styling**: Vanilla CSS (Tactile Void Design System)
- **Animation**: Framer Motion (State-driven transitions and scanner effects)
- **Icons**: Lucide React (Industrial icon set)
- **Infrastructure**: Vite (High-performance HMR)

## 🚀 Getting Started

1. **Clone the Dossier**:
   ```bash
   git clone https://github.com/sankhadipmondal05/EVAL.git
   ```

2. **Initialize Protocols**:
   ```bash
   npm install
   ```

3. **Deploy Locally**:
   ```bash
   npm run dev
   ```
